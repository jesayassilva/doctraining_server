API DocTraining

O DocTraining API é a principal forma para os aplicativos lerem os dados do DocTraining. É baseada em HTTP, portanto funciona com qualquer linguagem que tenha uma biblioteca HTTP, como cURL e urllib, ou então por serviços, como o $http do AngularJS. Todos as requisições são respondidas no formato JSON pela API.
