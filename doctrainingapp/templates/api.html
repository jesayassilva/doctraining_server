API do DocTraining

O DocTraining API é a principal forma para os aplicativos lerem os dados do DocTraining.
É baseada em HTTP, portanto funciona com qualquer linguagem que tenha uma biblioteca HTTP, como cURL e urllib, ou então por serviços, como o $http do AngularJS.
Todos as requisições são respondidas no formato JSON pela API.
Primeiro você precisa direcionar para URL do Servidor da API:

https://doctraining.herokuapp.com

A seguir as requisições disponíveis na API do DocTraining:

Salas

Para solicitar os nomes das salas disponíveis deve-se realizar uma solicitação pelo método na URL:

{% url 'doctrainingapp:todos_salas_api' %}

Se for bem-sucedida, essa chamada retornará uma lista com os id, nome da sala e quantidade de perguntas na sala .
A seguir um exemplo de resultado dessa requisição: 

[
    {
        "id": 5,
        "sala_nome": "Perguntas Gerais",
        "quantidade_perguntas": 12
    },
    {
        "id": 7,
        "sala_nome": "Perguntas Covid-19",
        "quantidade_perguntas": 50
    },
    {
        "id": 10,
        "sala_nome": "Perguntas sobre Influenza",
        "quantidade_perguntas": 30
    }
]

Perguntas de uma Sala

Para solicitar as perguntas de uma sala deve-se realizar uma solicitação pelo método usando o id da sala na URL:

api/salas/< int:pk_sala>/perguntas/

Se for bem-sucedida, essa chamada retornará uma lista com os id, pergunta, questão correta, questão incorreta 1, questão incorreta 2 e questão incorreta 3 da sala, seguindo a mesma ordem no JSON.

Amostra de Doença e Sintomas

Para solicitar uma amostra aleatória de uma doença e seus sintomas deve-se realizar uma solicitação pelo método na URL:

{% url 'doctrainingapp:um_caso_clinico_doenca_sintomas_api' %}

Se for bem-sucedida, essa chamada retornará id, nome da Doença e uma lista de sintomas.
A seguir um exemplo de resultado dessa requisição: 

{
    "id": 159,
    "doenca": "CONJUNTIVITE",
    "sintomas": [
        "ARDÊNCIA NOS OLHOS",
        "SENSAÇÃO ALGO NOS OLHOS",
        "LACRIMAÇÃO",
        "SECREÇÃO TRANSPARENTE NOS OLHOS",
        "OLHOS VERMELHOS"
    ]
}

Amostras de Todas as Doenças e Sintomas

Para solicitar todas as amostras das doenças e seus sintomas deve-se realizar uma solicitação pelo método na URL:

{% url 'doctrainingapp:todos_casos_clinicos_doencas_sintomas_api' %} /api/casos_clinicos/todos/

Se for bem-sucedida, essa chamada retornará uma lista no formato de id, nome da Doença e lista de sintomas.
